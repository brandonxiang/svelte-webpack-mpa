## Usage

To create a new project based on this template using degit:

```shell
npx degit brandonxiang/svelte-webpack-template svelte-app
cd svelte-app
yarn
```

## Development

```shell
yarn dev
# or
npm run dev
```

## Build

```shell
yarn build
# or
npm run build
```
